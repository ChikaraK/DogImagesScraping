# README
DogImagesScrapingはImagga Apiを用いて
画像解析を行うアプリケーションです。
当初の予定と変わり、犬以外にも一応の対応をしているため
正確にはImageScraperです！

使い方：
api_keyを環境変数で管理しているため、当アプリをお使いの場合は
１：https://imagga.com/　に登録
２：ルートディレクトリ下に.envファイルを製作の上
　　API_KEY = "人により"
　　API_SECRET = "違います"
の二つを定義して（ご自分のkeyに書き換えて）ご利用ください。
